Guess the Correct Color!

Color Box

Options

Score: 0

Instructions

Objective: Guess the correct color that matches the color displayed in the color box.

How to Play:

  1. Click on one of the color buttons below the color box.
  2. If you guess correctly, your score will increase, and a success message will appear.
  3. If you guess incorrectly, a message will prompt you to try again.
  4. The game will automatically start a new round after a correct guess.
